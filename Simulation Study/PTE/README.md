# PTE Simulation

This folder contains the simulation code for the probability treatment effect (PTE).

There are two main subfolders:

- `PointEstimation`: runs the point-estimation simulations and generates simulated datasets.
- `PTE inference`: runs the inference simulations using the datasets generated by `PointEstimation`.

Each subfolder contains its own README with more detailed instructions.

The results reported in Table S3.11 of the Supplementary Material are generated from these two folders.
