# Simulated Data Figures

This folder contains the code for generating figures from the simulated data-generating mechanism.

## Files

- `Figures.R`: generates the distribution plot.
- `Helper_functions.R`: contains the data-generating function and helper functions.

## Run

Run the script from this folder:

``` bash
Rscript Figures.R
```

## Output

The script generates:

``` text
FigureS3.1.pdf
```

This figure corresponds to Figure S3.1 in the Supplementary Material.
