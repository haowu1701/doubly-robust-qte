# Quantile Interpolation Figure

This folder contains the code for generating the quantile interpolation illustration.

## Files

- `New_quantile_figure.R`: generates the two-panel quantile interpolation figure.

This figure corresponds to Figure S2.1 in the Supplementary Material.

The left panel labeled `Yuqi JASA` refers to the quantile interpolation approach discussed in:

Tian, Y., Li, C., Tu, S., James, N. T., Harrell, F. E., & Shepherd, B. E. (2024). Addressing multiple detection limits with semiparametric cumulative probability models. DOI: <https://doi.org/10.1080/01621459.2024.2315667>
